Letakkan dua file bobot MobileNet di sini:

  model.json
  group1-shard1of1.bin

Cara dapat file ini: buka prefetch.html di komputer dengan internet, ikuti
langkah-langkahnya. Detail lengkap di README.md (di folder atas).

Selama folder ini kosong, app akan otomatis fallback ke download dari internet
saat pertama kali "Latih Model" dipakai.
